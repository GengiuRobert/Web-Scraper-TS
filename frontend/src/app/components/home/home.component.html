Web Scraper Application

This project is a full-stack web scraper built with Typescript, Node.js, MongoDB, and Angular18. The backend performs web scraping to extract book-related data from a website, which is then saved into a .json file. The MongoDB database is populated based on the data stored in this file, ensuring a structured and efficient data-handling process.

The scraping logic is handled by Puppeteer and Cheerio, with the extracted data saved in JSON format before being used to populate the database via a Node.js script. This allows for easy updates and modifications of the scraped data before insertion into MongoDB.

The frontend, developed with Angular18 and styled with Bootstrap, offers a simple, intuitive, and responsive user interface where users can easily view the scraped data, including book titles, authors, and descriptions. The use of Bootstrap ensures that the application looks great on various devices and screen sizes, enhancing usability and accessibility. It also implements Angular18 routing to allow smooth navigation between different sections of the application.
